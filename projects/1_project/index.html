  1. Camera Calibration (Direct Linear Transform) To convert 2D image coordinates into 3D world coordinates, the camera was calibrated using the Direct Linear Transform (DLT). A known set of 3D-2D correspondences was used to estimate the projection matrix P, allowing us to reconstruct 3D coordinates from a single image.

Key Steps

  • Capture known 3D object points and their 2D image projections.
  • Solve for the 3x4 projection matrix P using least squares.
  • Use P to map future 2D detections into 3D world space.
